jQuery EasyDropDown

A Drop-down Builder for Styleable Inputs and Menus

EasyDropDown turns the good old <select> element …

… into styleable drop-down menus for use in forms or general navigation.

It's like Chosen but with only the essential features:

  • Clean, semantic markup
  • Form compatibility
  • Keyboard control and textual search
  • Degradation to native UI on touch devices

Grab it on GitHub

To use, simply include the jquery.easydropdown.min.js script in your page and give your <select> tags the class dropdown:

<select class="dropdown">
	<option value="1">Option 1</option>
	<option value="2">Option 2</option>
	<option value="3">Option 3</option>
	<option value="4">Option 4</option>
</select>

No custom JS required!

Either create your own styles to match your design and branding, or use one of our ready made themes.

Each drop-down has the following markup structure that you can target with your CSS:

<div class="dropdown">
	<select>...</select>
	<span class="selected">Option 1</span>
	<span class="carat"></span>
	<div>
		<ul>
			<li>Option 1</li>
			<li>Option 2</li>
			<li>Option 3</li>
			<li>Option 4</li>
		</ul>
	</div>
</div>
